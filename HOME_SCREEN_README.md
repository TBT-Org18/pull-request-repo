# Contoso IT Solutions - Home Screen

A modern, minimalist home screen website for Contoso IT Solutions, showcasing their enterprise IT services.

## 🎨 Features

- **Modern & Minimalist Design**: Clean, professional appearance with excellent use of whitespace
- **Fully Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Service Showcase**: Highlights 6 key IT services with icons and descriptions
- **Smooth Interactions**: Smooth scrolling navigation and interactive elements
- **Contact Form**: Easy-to-use contact form for customer inquiries
- **About Section**: Company overview with key statistics
- **Sticky Navigation**: Easy access to all sections from anywhere on the page

## 📁 Files Included

- **index.html** - Main HTML structure
- **styles.css** - Complete styling with responsive design
- **script.js** - Interactive features and smooth scrolling

## 🚀 Getting Started

1. Open `index.html` in any modern web browser
2. No additional dependencies or build tools required
3. Works entirely offline

## 📋 Sections

1. **Header** - Navigation bar with logo and links
2. **Hero** - Main banner with call-to-action button
3. **Services** - Grid showcasing 6 IT services:
   - Cloud Solutions
   - Cybersecurity
   - IT Consulting
   - Support & Maintenance
   - Data Analytics
   - Network Solutions
4. **About** - Company information with key metrics
5. **Contact** - Contact information and inquiry form
6. **Footer** - Copyright and links

## 🛠️ Customization

You can easily customize:
- Company name and contact information in `index.html`
